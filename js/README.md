# Facturation Atelier (version régénérée)

Cette version est volontairement **simple et stable** (sans Service Worker) pour éviter les problèmes de cache.

## Publier sur GitHub Pages
1. Déposer tous les fichiers à la racine du dépôt (branche `main`).
2. GitHub → **Settings** → **Pages**
3. Source : **Deploy from a branch**
4. Branch : `main` / folder : `/ (root)` puis **Save**
5. Ouvrir l’URL GitHub Pages.

## Forcer un rechargement (si besoin)
- Mac / Chrome : `Cmd + Shift + R`
- iPhone : actualiser + fermer/réouvrir l’onglet si nécessaire
